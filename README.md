# Todo list app

A todo list app

### General steps to follow when creating a project

- Thinking about the UI
- Functionality
  - Add a TODO
  - Delete a TODO
  - check task
  - total items count
  - Select all task
  - clear all task
    -Sort
    -All list
    -Uncomplete
    -Complete
- Data
  - tasks - an array
    -localstorage
  - task - {done, text, id}
- Functions (in code)
  - addTodo
  - deleteTodo
    -deleteAllTodo
  - checkTodo
  - showTodosList
  - showNotification if not enter string or text
